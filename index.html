What's RescuePH?
RescuePH started as a #rescuePH in Twitter and a Google Docs Spreadsheet. Initiated by JP Loh, Ros Juan, and Tonyo Cruz as a way to collate and report important rescue information open to the public. Netizens then type the status of the rescue alert, location, details of of the report, source, and the exact tweet to a Spreadsheet row. That spreadsheet is then monitored by the RescuePH Team then repacks that data for local rescue units.
What does this site do?
Maps Integration
We have Maps in our website that contains pins of people that need rescue.
Twitter Integrated.
We get tweets with #rescuePH. No need to manually search for them.
Twitter and Submission Form Side by Side.
No need to have Twitter open on one tab and the old Google docs on the other.
These 2 are all in one page so that volunteer work doesn't encounter server overloading.
Responsive.
This site wants to be responsive. Being responsive means being responsible.
